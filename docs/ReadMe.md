# SONEX
Sonex is currently in private beta, it is not expected to work on systems not expressly configured to use it. That being said if you want to "build" yourself a copy feel free.
# Get it Yourself
External Dependancies:
- [Conda-like package manager (prefer mamba)](https://docs.conda.io/projects/conda/en/latest/user-guide/install/index.html)
  - [Montreal Forced Aligner](https://github.com/MontrealCorpusTools/Montreal-Forced-Aligner)
- [Piplike](https://github.com/pypa/pip) [manager](https://peps.python.org/pep-0508/) (prefer [UV](https://docs.astral.sh/uv/))
  - Install requirements.txt
- [FFMPEG](https://www.ffmpeg.org/)
- git clone / wget this repo
- installer/sonex.py
